📖 Understanding the Magic Number Method

1 Find the Interesting Octet: This is the octet in the subnet mask that's not 255 or 0.
2 Calculate the Magic Number: Magic Number = 256 - (Interesting Octet Value)
3 Find Subnet Boundaries: Start from 0 and add the magic number repeatedly.
4 Identify Your Subnet: Find which range your IP falls into.

Example: IP 192.168.10.45 with mask 255.255.255.192

  • ✓ Interesting octet: 4th octet (192)
  • ✓ Magic Number: 256 - 192 = 64
  • ✓ Subnets: 0, 64, 128, 192...
  • ✓ IP .45 falls in the 0-63 range
  • ✓ Network: 192.168.10.0, Broadcast: 192.168.10.63
